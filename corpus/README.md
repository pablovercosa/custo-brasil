# Custo Brasil — Corpus Público

Conversões públicas de documentação técnica do Sistema Nacional de Documentos Fiscais Eletrônicos.

## Estrutura

- **nfe/** — NF-e (modelo 55): notas técnicas, manuais, informes técnicos, schemas XSD
- **nfce/** — NFC-e (modelo 65): notas técnicas, manuais, informes técnicos, schemas XSD
- **nfag/** — NFAg (Água e Saneamento): manuais, schemas XSD
- **nfeabi/** — NFeABI: manuais
- **nfgas/** — NFGas (Gás): manuais, schemas XSD
- **combustiveis/** — Tributação monofásica de combustíveis: notas técnicas, informes técnicos
- **reforma-tributaria/** — IBS/CBS/Reforma Tributária: notas técnicas, informes técnicos, schemas XSD
- **geral/** — Tabelas transversais, documentos multi-DFe: notas técnicas, informes técnicos, tabelas federais, schemas XSD
- **cte/**, **mdfe/**, **nfse/** — Reservado para expansão futura

## Sobre

Convertido com Docling Server a partir de PDFs originais do Portal Nacional da NF-e e fontes estaduais.

Imagens referenciadas em `assets/` (sem Base64). Links relativos compatíveis com Obsidian e GitHub.
