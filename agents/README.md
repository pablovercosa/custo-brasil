# Agents Públicos — Custo Brasil

Agents especializados para consulta e análise do corpus público de documentos fiscais brasileiros.

## Agents disponíveis

| Agent | Descrição |
|-------|-----------|
| `pesquisador-fiscal` | Localiza e explica evidência normativa no corpus |
| `auditor-diff-regulatorio` | Compara versões de documentos normativos |
| `explicador-rejeicao` | Explica códigos de rejeição SEFAZ com base nas regras de validação |
| `analisador-schema` | Interpreta metadados de schemas XSD |
| `especialista-reforma-tributaria` | Consulta especializada sobre IBS/CBS |

## Base de conhecimento compartilhada

Consulte `CORPUS.md` para a referência completa da estrutura do corpus, campos de front matter, categorias, convenção de slugs e localização de documentos.

## Skills complementares

Skills públicas em `skills/` fornecem passo-a-passo operacionais para cada domínio de consulta. Consulte `skills/README.md` para a lista completa.
