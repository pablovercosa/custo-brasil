## Detalhamento por UF

### Região Norte

- **AC** — Interna 18%. Sem majorações gerais conhecidas.
- **AP** — Interna 18%. Alíquota única para a maioria das operações.
- **AM** — Interna 18%. ZFM com benefícios diferenciados (crédito presumido).
- **PA** — Interna 18%. Alíquotas reduzidas para alguns produtos da cesta básica.
- **RO** — Interna 17,5%. Peculiaridade: alíquota com fração.
- **RR** — Interna 17%.
- **TO** — Interna 18%.

### Região Nordeste

- **AL, BA, CE, MA, PB, PE, PI, RN, SE** — Interna 18% na maioria das operações.
- **PE** — Alíquotas internas diferenciadas para alguns setores (ex: 17% para alguns produtos).

### Região Centro-Oeste

- **DF** — Interna 18%.
- **GO** — Interna 17%.
- **MT** — Interna 17%. Alíquotas diferenciadas para agronegócio.
- **MS** — Interna 17%.

### Região Sudeste

- **ES** — Interna 17%.
- **MG** — Interna 18%. Alíquotas reduzidas para cesta básica.
- **RJ** — Interna 20% (desde 2016, Decreto 46.409/2018). Uma das mais altas do país.
- **SP** — Interna 18%. Alíquota de 12% para alguns produtos (ex: alguns alimentos).

### Região Sul

- **PR** — Interna 19% para a maioria das operações; 18% para alguns setores.
- **RS** — Interna 17% (padrão); 18% para algumas operações.
- **SC** — Interna 17%.

## FCP por UF

A alíquota do FCP é de até **2%** na operação interna. A maioria dos estados adota 2%. A tabela completa e oficial está em `corpus/geral/tabelas-federais/tabela-fcp-uf-2025/`.

> **Aviso**: As alíquotas podem ser alteradas por lei estadual a qualquer tempo. Consulte a legislação vigente de cada UF para cálculos reais. Produtos específicos (energia elétrica, telecomunicações, combustíveis, etc.) podem ter alíquotas diferentes da padrão.
