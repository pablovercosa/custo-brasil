# Skills Públicas — Custo Brasil

Skills operacionais em formato `SKILL.md` com frontmatter YAML para consulta e análise do corpus.

## Skills disponíveis

| Skill | Descrição |
|-------|-----------|
| `comparar-notas-tecnicas` | Compara versões de notas técnicas ou manuais |
| `consultar-regra-fiscal` | Consulta base normativa para regras fiscais |
| `explicar-rejeicao-sefaz` | Explica códigos de rejeição SEFAZ |
| `localizar-regra-vigente` | Localiza versão aplicável para uma data |
| `mapear-campo-xml` | Localiza metadados de campos XML em schemas |
| `reforma-tributaria` | Consulta especializada IBS/CBS |
| `navegar-corpus` | Guia de navegação e descoberta de documentos |

## Agents complementares

Agents públicos em `agents/` fornecem visão mais analítica para cada domínio. Consulte `agents/README.md` para a lista completa.
